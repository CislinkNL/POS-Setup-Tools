## 🔧 常用工具快捷方式

```batch
# Windows 系统工具
Win + R → netplwiz          # 用户管理
Win + R → services.msc      # 服务管理
Win + R → ncpa.cpl          # 网络连接
Win + R → control printers  # 打印机设置
Win + R → intl.cpl          # 区域设置
Win + R → firewall.cpl      # 防火墙设置

# POS 专用脚本
Enable_Settings.bat         # 修复 Settings
Install_RustDesk_Service.bat # 安装远程管理
POS_Setup.bat              # 完整 POS 配置
POS_restore.bat            # 恢复默认设置
```
